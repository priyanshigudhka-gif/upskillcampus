# 🌾 Agriculture Crop Yield Prediction

## 📌 Project Overview
This project predicts crop yield (Quintal per Hectare) using Machine Learning based on factors like crop type, state, cost of cultivation, and cost of production. It helps in better decision-making for farmers and agricultural planning.

---

## 🚀 Live Application
👉 https://upskillcampus-6yzgtgkpgctwzx4eaef72f.streamlit.app/

---

## 🧠 Machine Learning Model
- Model Used: Random Forest Regressor
- Problem Type: Regression
- Target Variable: Yield (Quintal per Hectare)

---

## 📊 Features
- Crop-wise yield prediction
- State-wise yield prediction
- Cost-based prediction (A2+FL, C2, Production Cost)
- Interactive Streamlit web application
- Feature importance visualization

---
